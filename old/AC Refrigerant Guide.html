🌡️ Superheat — R-134a

📖 How to Measure Superheat

1. Connect gauge to suction port (low side).
2. Read pressure → use PT chart to find saturation temp.
3. Measure suction line temp at evaporator outlet (pipe-clamp thermometer).
4. Superheat = Line Temp − Saturation Temp.

Target: See per-refrigerant target below (typically 5–15°F / 3–8°C). Too low = overcharged. Too high = undercharged.

❄️ Subcooling — R-134a

ℹ️ Fixed Orifice / Piston System

Your system uses a fixed orifice or piston — not a TXV/EEV. Use the Superheat method (above) to check charge. Subcooling is only applicable to TXV/EEV systems.

Saturation Temp − Actual Liquid Line Temp = Subcooling. For TXV / EEV systems.

📖 How to Measure Subcooling

1. Connect gauge to liquid line port (high side).
2. Read pressure → use PT chart to find saturation temp.
3. Measure liquid line temp at condenser outlet.
4. Subcooling = Saturation Temp − Line Temp.

Target: See per-refrigerant target below (typically 5–12°F / 3–7°C). Too low = undercharged. Too high = overcharged.

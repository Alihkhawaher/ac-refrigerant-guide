# AC Refrigerant Quick Reference — T3 Climate (Middle East / Gulf)

A quick-reference tool for automotive and residential AC refrigerant pressures, targeting **T3 climate conditions** common in the Middle East and Gulf region (ambient temperatures up to 55°C / 131°F).

## What This Is

- **Interactive HTML guide** (`AC Refrigerant Guide.html`) with refrigerant selection, pressure ranges, P-T charts, system diagrams, superheat/subcooling diagnostics, and a bilingual English/Arabic interface
- **Python validator** (`ac_calculator.py`) that verifies all P-T data, pressure ranges, and calculations

## Refrigerants Covered

| Refrigerant | Application |
|---|---|
| R-134a | Automotive AC (1994–2020) |
| R-22 | Legacy residential AC (banned 2020) |
| R-410A | Residential AC, Heat Pumps |
| R-32 | Modern residential AC |

## Important Notes

- **Fixed orifice / capillary tube systems only** — this guide does **not** cover TXV/EEV (expansion valve) systems
- **T3 climate focus** — operating pressure ranges reflect extreme ambient conditions (up to 55°C / 131°F) typical of Saudi Arabia, UAE, Kuwait, Qatar, Bahrain, and Oman
- Pressure data sourced from hvacptcharts.com, acdirect.com, Chemours/Opteon, and manufacturer reference charts
